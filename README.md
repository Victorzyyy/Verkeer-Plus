# Verkeersplus — Next.js Website

Production-ready website for Verkeersplus Services B.V.

## Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **next/font** (Inter, Space Grotesk, Space Mono, Noto Sans Arabic)
- **next/image** (optimized images)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Language Toggle

The site ships with a Dutch/Arabic bilingual toggle in the header. All copy lives in `src/data/content.ts` under `ui.nl` and `ui.ar`. The `useT()` hook returns the right language object based on the current context.

RTL layout is handled automatically:
- `<html lang="ar" dir="rtl">` is set when Arabic is active
- Tailwind RTL utilities and custom CSS handle mirroring
- Arabic text uses Noto Sans Arabic (loaded via next/font)

## Sitemap & SEO

- `/sitemap.xml` — generated by `src/app/sitemap.ts`, includes hreflang for NL and AR
- `/robots.txt` — generated by `src/app/robots.ts`
- Each page has `metadata` export with `title`, `description`, `openGraph`, and `alternates`

## Adding Content

Edit `src/data/content.ts`:
- `services[]` — service tab content
- `trustItems[]` — Over Ons bullet points
- `contactInfo[]` — phone, email, etc.
- `fitItems[]` — Werken bij sidebar checklist
- `ui.nl` / `ui.ar` — all other UI copy

## Forms

Both forms (`WerkenBijForm` and `ContactForm`) use a simulated `setTimeout` for submission. In production, replace with:

```typescript
const res = await fetch('/api/sollicitatie', {
  method: 'POST',
  body: formData,
})
```

Add `src/app/api/sollicitatie/route.ts` for the backend handler (Nodemailer, Resend, etc.).

## Deployment

Deploy to Vercel (recommended) or any Node.js host:

```bash
npm run build
npm start
```

Set `NEXT_PUBLIC_SITE_URL` env var to your domain and update `siteConfig.baseUrl` in `src/data/content.ts`.
